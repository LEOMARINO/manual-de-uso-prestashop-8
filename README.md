# Guía del usuario

PrestaShop presenta un panel de administración de usuarios completo e intuitivo y le ofrece cientos de funciones estándar que pueden adaptarse o personalizarse para responder a todas sus necesidades.

Esta guía del usuario le ayudará a familiarizarse con todas las funciones de PrestaShop. También podrá administrar eficientemente su sitio PrestaShop. Si bien la mayor parte está dirigida a propietarios de tiendas, el primer capítulo sirve como una introducción a la interfaz de la oficina principal, que puede resultar útil para todos.

Puede publicar todas sus preguntas directamente en nuestro foro: [http://www.prestashop.com/forums/](http://www.prestashop.com/forums/)
