# Gestionando a tus clientes

Debe cuidar bien a sus clientes y comienza asegurándose de que su perfil contenga toda la información que necesita para enviarles un paquete.

El menú "Clientes" le permite consultar los datos y direcciones de sus clientes.

La configuración para todo lo relacionado con el cliente se encuentra en el menú "Parámetros de la tienda", en la página "Configuración del cliente".

Este capítulo contiene las siguientes secciones:

* [Tus clientes](your-customers.md)
* [Direcciones de clientes](customer-addresses.md)
* [Sobresaliente](outstanding.md)
