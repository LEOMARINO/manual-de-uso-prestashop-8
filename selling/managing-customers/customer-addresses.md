# Direcciones de clientes

Al hacer clic en la página "Direcciones", tendrá acceso a la lista de direcciones de sus clientes (casa, trabajo, etc.). Puede editarlos usando el botón "Editar" o eliminarlos por completo.

![](<../../.gitbook/assets/51839324 (3) (3) (3).png>)

Debajo de la tabla de direcciones se encuentra el botón "Establecer campos obligatorios para esta sección". Abre un formulario donde puedes indicar si un campo de base de datos es necesario o no marcando las casillas correspondientes: de esta manera, puedes hacer que campos como "empresa" o "número de teléfono" sean obligatorios cuando un cliente registra una nueva dirección en tu tienda.

![](<../../.gitbook/assets/51839325 (3) (3) (3).png>)

## Agregar una nueva dirección <a href="#customeraddresses-addinganewaddress" id="customeraddresses-addinganewaddress"></a>

Tienes la posibilidad de crear tú mismo nuevas direcciones para un cliente con el botón "Agregar nueva dirección". Abre un formulario con el tipo de campo que cabría esperar.

![](<../../.gitbook/assets/51839326 (3) (3) (1).png>)

Algunos campos, sin embargo, requieren cuidados especiales:

* **Correo electrónico del cliente**. Este campo es esencial: si está agregando una dirección para un cliente existente, DEBE identificar a este cliente utilizando su dirección de correo electrónico. De lo contrario, PrestaShop no sabrá a quién asignarle esta dirección.
* **Número de identificación**. El número del DNI de esta persona, o un número de identificación fiscal único. Por supuesto, este campo es opcional.
* **Alias ​​de dirección**. Una breve descripción, con el fin de ayudar al cliente a elegir la dirección correcta: "Casa", "Oficina", "Casa de la tía Beth", etc.
* **Compañía**. El nombre de la empresa del cliente, si es necesario.
* **Número de IVA**. El número de identificación de IVA del cliente (o de la empresa del cliente).
* **Otro**. Alguna información adicional que pueda ser útil para el envío.
