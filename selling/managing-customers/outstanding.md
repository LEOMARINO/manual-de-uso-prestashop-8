# Pendiente

Esta página solo está disponible cuando habilita las funciones B2B de PrestaShop. Para hacerlo, vaya a la página "Configuración del cliente" y elija "Sí" para la opción "Habilitar modo B2B".

Esta página muestra el informe pendiente de pago de algunos de sus clientes, es decir, la cantidad de dinero que les permite comprar sus productos antes de que tengan que pagarle.

## Habilitación de la característica excepcional <a href="#outstanding-enablingtheoutstandingfeature" id="outstanding-enablingtheoutstandingfeature"></a>

Cuando el modo B2B está habilitado (página Parámetros de la tienda > Configuración de clientes), se agregan tres campos al formulario de creación de clientes:

![](<../../.gitbook/assets/51839331 (3) (3).png>)

* **Monto pendiente permitido**. La cantidad máxima de dinero que este cliente puede utilizar como pendiente.
* **Número máximo de días de pago**. El número máximo de días que el cliente puede pasar antes de tener que pagar su pendiente.
* **Calificación de riesgo**. El factor de riesgo para este cliente: Ninguno, Bajo, Medio o Alto. Depende de usted decidir si un cliente es un benefactor arriesgado de su excelente característica o si seguramente pagará a tiempo.

Debe completar estos campos para todos los clientes profesionales.

## Pantalla pendiente actual <a href="#outstanding-currentoutstandingscreen" id="outstanding-currentoutstandingscreen"></a>

Una vez que un cliente con asignación pendiente compra en su tienda, aparece en la página Clientes > Pendientes. Esto le permite obtener una descripción general de sus facturas pendientes actuales.
