# Gestión de existencias

* &#x20;[Resumen de existencias](stock-overview.md)
* &#x20;[Movimientos de acciones](stock-movements.md)

Puedes gestionar el stock de tu tienda desde la página "Stock", en el menú "Catálogo". No se requiere configuración para activar la gestión de stock.

La página de gestión de stock consta de dos pestañas: "Stock" y "Movimientos".

De forma predeterminada, cuando abre la página de gestión de stock, la pestaña "Stock" es la pestaña abierta. Dentro de esta pestaña podrás tener una visión general del stock de productos de tu catálogo y gestionar sus cantidades. La pestaña "Movimientos" le ofrece un historial de todos los cambios en las cantidades.
