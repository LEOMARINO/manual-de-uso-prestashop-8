# Gestionar el servicio al cliente

![](<../../.gitbook/assets/51839867 (3) (3) (2).png>)

En PrestaShop 1.7, las secciones de Atención al Cliente obtienen un menú dedicado para darle la importancia que se merece. Manejar el servicio de atención al cliente es clave para su éxito, por lo que estas páginas le permiten dar seguimiento fácilmente a las solicitudes de soporte y devoluciones de mercancías de sus clientes.&#x20;
