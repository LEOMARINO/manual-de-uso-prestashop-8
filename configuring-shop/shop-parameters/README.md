# Configurando los parámetros de tu tienda

Los parámetros de la tienda se centran en la experiencia de compra del cliente. Aquí es donde definirás todas las opciones respecto a tus clientes, los pedidos y el comportamiento general de tu tienda. Consulta cada una de estas páginas antes de lanzar tu tienda: hay muchas decisiones que debes tomar antes de lanzarla y cada una de ellas puede afectar la experiencia de tus clientes de alguna manera.

Este capítulo contiene las siguientes secciones:

* [Configuración de pedido](order-settings/)
* [Configuración del producto](product-settings.md)
* [Configuración del cliente](customer-settings/)
* [Contact](contact.md)
* [Traffic](traffic/)
