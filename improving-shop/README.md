# Mejorando tu tienda

A medida que su tienda y su negocio evolucionen, querrá agregar nuevas funcionalidades o explorar nuevos mercados. Esta sección se trata de personalizar tu tienda: darle forma a una tienda en línea que se adapte a tu negocio actual y escalarla con nuevos módulos, nuevos países, idiomas, métodos de pago o transportistas, a medida que pasa el tiempo.
