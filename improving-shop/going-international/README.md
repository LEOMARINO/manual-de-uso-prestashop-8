# Going International

Por muy obvio que parezca, esto es Internet, y gente de todo el mundo seguramente visitará su tienda y pedirá sus productos. Como propietario de una tienda, debes atender a la mayor cantidad de clientes posible. Esto significa tener tu sitio web completamente traducido a tantos idiomas como sea necesario, con impuestos locales, pesos y unidades monetarias, zonas geográficas locales, etc.
