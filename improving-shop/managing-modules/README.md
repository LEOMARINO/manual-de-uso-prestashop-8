# Administrar sus módulos

PrestaShop viene con un conjunto de módulos que se pueden instalar/desinstalar y configurar como se desee, para personalizar y completar su tienda.

La gama de acciones es prácticamente ilimitada: la extensibilidad de PrestaShop permite convertir su tienda exactamente en lo que usted desea, en lugar de tener que cumplir con restricciones que no eligió, siempre que pueda encontrar el módulo que hace exactamente lo que necesita.

Este capítulo contiene las siguientes secciones:

* [Módulos y servicios](modules-and-services/)
* [Catálogo de módulos](broken-reference)
