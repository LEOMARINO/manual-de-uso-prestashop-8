# Administrar varias tiendas

La función multitienda le permite administrar varias oficinas principales desde una única oficina administrativa.&#x20;

Esta característica es especialmente útil si te encuentras en uno de los siguientes casos:

* **Quieres establecer un nombre de dominio diferente para cada una de tus tiendas.**\

* **Quieres tener una versión B2B y B2C de tu tienda.**\

* **Quieres que tus tiendas tengan marcas y diseños diferentes.**\

* **Quieres vender los mismos productos a un precio diferente en cada una de tus tiendas.**

## Habilitación de la función multitienda <a href="#managingmultipleshops-enablingthemultistorefeature" id="managingmultipleshops-enablingthemultistorefeature"></a>

2. Habilite la función multitienda y guarde.
3. La página "Multitienda" ahora está disponible en el menú **Parámetros avanzados**.
