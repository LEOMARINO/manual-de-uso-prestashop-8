# Agregar un nuevo grupo de tiendas

Cuando la función multitienda está habilitada, un selector desplegable (o el encabezado, a partir de la versión 1.7.8.0) le permite seleccionar el contexto en el que desea trabajar (Todas las tiendas, grupo de tiendas o tienda única). Así, definir grupos de tiendas te permitirá aplicar cambios a todas las tiendas de un grupo a la vez. Es muy útil si algunas de tus tiendas comparten la mayoría de las características _(catálogo, empleados, transportistas, etc.)._&#x20;

{% hint style="info" %}
La configuración de las tiendas de un grupo también se puede personalizar individualmente si es necesario. Todo lo que tienes que hacer es cambiar a un contexto de tienda única. &#x20;
{% endhint %}

Para agregar un nuevo grupo de tiendas, vaya a la página **Parámetros avanzados > Multitienda**, haga clic en el botón **"Agregar un nuevo grupo de tiendas"** y complete el formulario.

<figure><img src="../../.gitbook/assets/image (25) (2).png" alt=""><figcaption></figcaption></figure>

* **Nombre del grupo de tiendas:** Ingrese un nombre para este grupo de tiendas. Este nombre es privado: sólo tú y tu equipo podréis verlo. Puedes editarlo en cualquier momento.
* **Color:** También puedes establecer un color diferente para cada grupo de tiendas. Dependiendo del grupo en el que estés trabajando, el encabezado cambiará de color. Esto te permite saber de un vistazo en qué contexto te encuentras.
* **Compartir clientes:** Habilite esta función si desea permitir que sus clientes utilicen la misma dirección de correo electrónico y contraseña para iniciar sesión en todas las tiendas de un grupo.

{% hint style="warning" %}
No podrá desactivar esta función una vez que haya registrado clientes.&#x20;
{% endhint %}

* **Compartir cantidades disponibles para vender:** Habilite esta función si desea que los productos idénticos vendidos en las tiendas del grupo tengan las mismas cantidades disponibles.&#x20;
* **Compartir pedidos:** Esta opción solo se puede habilitar si las opciones "Compartir clientes" y "Compartir cantidades disponibles para vender" están habilitadas. Si esta función está habilitada, los clientes que iniciaron sesión y agregaron artículos a su carrito podrán completar su pedido en cualquier otra tienda del grupo.

{% hint style="warning" %}
No podrás desactivar esta función una vez que hayas aceptado los pedidos.&#x20;
{% endhint %}

* **Estado:** Puedes habilitar o deshabilitar un grupo de tiendas en cualquier momento.
